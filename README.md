# YuanJing Ye ePortfolio

A static, responsive ePortfolio website designed for GitHub Pages.

## Publish on GitHub Pages

1. Create a new GitHub repository named `yuanjing-ye-portfolio`.
2. Upload these files and the CV to `assets/My-CV.pdf`.
3. In the repository, go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder, then save.
6. GitHub will show the public site address after deployment.

## Before publishing

- Put the CV file at `assets/My-CV.pdf`.
- Replace the inactive LinkedIn placeholder in `index.html` with the real profile URL.
- Review all personal reflections and peer-feedback text for accuracy.
